## 🚀 使用指南

### 1. 基础使用
```bash
# 运行演示程序
cd crates/itools
cargo run --example mcp_demo

# 构建项目
cargo build
```

### 2. 在 SeeU Desktop 中使用
- 在 iTools 界面中点击 "⚙️ MCP设置" 选项卡
- 使用模板快速创建服务器配置
- 监控服务器性能和健康状态
- 执行批量操作管理多个服务器

### 3. 配置文件位置
```
~/.config/seeu_desktop/mcp_servers.json
```
