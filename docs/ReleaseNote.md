# SeeU Desktop 发布说明

## 🎨 v0.3.712 (2025-01-13) - 动画启动页面更新

### ✨ 新增功能
- **🎬 动画启动页面**: 全新设计的SVG动画启动页面，提升用户体验
  - **中央Logo动画**: 脉冲效果展示应用品牌
  - **功能模块展示**: 5个核心功能模块（笔记、搜索、终端、文件、AI）依次出现
  - **连接线动画**: 动态绘制模块间的连接关系
  - **渐变进度条**: 带有移动高光效果的进度指示器
  - **打字机效果**: 应用标语逐字显示动画
  - **深蓝渐变背景**: 现代化的视觉设计风格

### 🔧 技术改进
- **中文字符处理**: 修复启动页面中文字符边界处理问题
- **动画性能优化**: 使用时间驱动的动画系统，确保流畅体验
- **响应式布局**: 动画元素自适应不同屏幕尺寸
- **🚀 启动性能大幅优化**: 从6秒延迟优化到几乎瞬时启动
  - **数据库快速初始化**: 延迟创建FTS5全文搜索索引，优先创建基础表结构
  - **SQLite性能优化**: 启用WAL模式、内存缓存等优化设置
  - **搜索索引延迟加载**: 搜索索引在后台延迟500ms初始化，不阻塞UI
  - **启动进度优化**: 目标时间从2秒减少到1.2秒，更快的用户反馈
  - **字体缓存优化**: 从启动时同步加载改为2秒后后台加载，字体验证延迟执行
  - **网络请求优化**: 连接超时从30秒减少到2秒，总超时减少到5秒
  - **插件系统优化**: WASM运行时按需初始化，插件市场延迟刷新

### 🎯 用户体验提升
- 替换单调的进度条为丰富的动画效果
- 通过视觉动画展示应用的核心功能定位
- 提升首次使用时的品牌印象和专业感

## 📋 v0.2.0 发布说明

## 🚀 重大更新

欢迎使用 SeeU Desktop v0.2.0！这是一个包含重大功能更新的版本，新增了安全终端、MCP协议集成、AI助手工具调用等核心功能，大幅提升了应用的实用性和扩展性。

## 📅 发布信息

- **版本号**: v0.2.0
- **发布日期**: 2024年12月
- **技术栈**: Rust + egui + eframe
- **支持平台**: Windows, macOS, Linux

## ✨ 新增功能

### 🖥 iTerminal - 安全终端模块 (新增)
- **多标签页终端**: 支持同时运行多个终端会话
- **23个安全命令**: 涵盖文件操作、系统信息、搜索等常用功能
- **命令历史管理**: 保存、搜索和导航历史命令
- **异步命令执行**: 不阻塞UI的命令处理机制
- **安全沙箱**: 只允许执行预定义的安全命令
- **丰富配置**: 字体、颜色、行为等全面可配置
- **🆕 远程服务器管理**: 一键连接SSH远程服务器
  - 支持密码、私钥、SSH Agent三种认证方式
  - 安全的敏感信息加密存储
  - 服务器配置导入/导出功能
  - 连接测试和状态监控
  - 标签分类和搜索过滤

### 🔗 MCP协议集成 (新增)
- **完整MCP支持**: 基于Model Context Protocol的标准实现
- **JSON-RPC 2.0**: 支持标准的JSON-RPC 2.0通信协议
- **多传输方式**: 支持stdio和SSE传输类型
- **服务器管理**: 完整的MCP服务器生命周期管理
- **预设模板**: 7个分类、7个预设服务器模板
- **状态监控**: 实时监控服务器连接和健康状态
- **批量操作**: 支持批量启动、停止、测试服务器

### 🤖 AI助手工具调用 (新增)
- **MCP工具集成**: AI助手可调用MCP服务器提供的工具
- **工具调用确认**: 用户手动确认工具调用，确保安全性
- **自动反馈**: 工具执行结果自动发送给AI获取智能反馈
- **完整日志**: 详细的工具调用和执行日志
- **多服务器支持**: 支持选择不同的MCP服务器
- **流式处理**: 支持流式和非流式两种处理模式

## 📦 安装要求

### 系统要求
- **Windows**: Windows 10 或更高版本
- **macOS**: macOS 10.15 或更高版本  
- **Linux**: 支持主流发行版（Ubuntu 20.04+, CentOS 8+等）

### 硬件要求
- **内存**: 最低 4GB RAM，推荐 8GB 或更多
- **存储**: 最低 500MB 可用空间
- **处理器**: 64位处理器
