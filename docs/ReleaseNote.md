# SeeU Desktop v0.1.0 发布说明

## 🎉 首次发布

欢迎使用 SeeU Desktop v0.1.0！这是我们基于 Rust + egui + eframe 技术栈构建的现代化桌面应用程序的首个正式版本。

## 📅 发布信息

- **版本号**: v0.1.0
- **发布日期**: 2024年12月
- **技术栈**: Rust + egui + eframe
- **支持平台**: Windows, macOS, Linux

## ✨ 核心特性

### 🏠 主页功能
- **默认启动页面**: 应用启动时显示功能概览和快速入口
- **功能卡片**: 智能笔记、全局搜索、AI工具集的直观展示
- **系统状态**: 实时显示CPU、内存使用率和数据统计
- **快速操作**: 一键创建笔记、开始搜索、管理插件

### 📝 iNote - 智能笔记模块
- **笔记本管理**: 树形结构的笔记本组织系统
- **Markdown支持**: 完整的Markdown编辑和预览功能
- **富文本编辑**: 支持格式化文本编辑
- **标签系统**: 灵活的标签管理和分类
- **附件支持**: 图片、文件等附件管理
- **思源笔记导入**: 支持从思源笔记导入数据
- **自动保存**: 实时保存编辑内容
- **SQLite存储**: 高性能的本地数据库存储

### 🔍 iSearch - 全局搜索模块
- **文件索引**: 高效的文件系统索引和搜索
- **多格式支持**: 支持PDF、DOC、TXT、MD等多种文件格式
- **内容搜索**: 文件内容全文搜索
- **高级语法**: 支持复合查询和过滤条件
- **实时预览**: 搜索结果内容预览
- **关键词高亮**: 搜索结果中关键词突出显示
- **目录管理**: 灵活的索引目录添加和管理
- **性能优化**: 缓存机制和增量索引

### 🔧 iTools - AI工具集成模块
- **多角色权限**: 业务用户、开发者、运维人员、管理员四种角色
- **插件市场**: 浏览、搜索、安装AI工具插件
- **MCP协议**: 基于Model Context Protocol的插件通信
- **安全沙箱**: 插件安全执行环境
- **预置插件**: 文件系统、Git集成、BI连接器等

### 🤖 AI助手功能
- **智能对话**: 支持自然语言交互
- **历史会话**: 持久化的聊天记录管理
- **模型选择**: 支持多种AI模型切换
- **OpenAI兼容**: 支持OpenAI兼容的API接口
- **附件支持**: 图片、文件、代码片段上传
- **命令菜单**: @命令快速调用功能

### 🎨 用户界面
- **现代化设计**: 基于egui的即时模式GUI
- **主题支持**: 亮色/暗色主题切换
- **响应式布局**: 自适应窗口大小的界面设计
- **模块化导航**: 清晰的左侧导航栏
- **状态栏**: 实时系统资源监控
- **右侧边栏**: 可折叠的AI助手面板

### 🔧 系统功能
- **跨平台**: Windows、macOS、Linux原生支持
- **高性能**: 纯Rust实现，低资源占用
- **配置管理**: 灵活的应用设置和配置
- **窗口状态**: 自动保存和恢复窗口位置大小
- **启动优化**: 异步加载和懒加载机制
- **日志系统**: 完整的应用日志记录

## 🚀 性能特点

- **启动速度**: 优化的启动流程，快速响应
- **内存占用**: 相比混合架构应用，内存使用更少
- **渲染效率**: egui即时模式GUI，高效渲染
- **搜索性能**: 优化的索引算法，快速搜索响应
- **数据存储**: SQLite本地数据库，高效数据管理

## 📦 安装要求

### 系统要求
- **Windows**: Windows 10 或更高版本
- **macOS**: macOS 10.15 或更高版本  
- **Linux**: 支持主流发行版（Ubuntu 20.04+, CentOS 8+等）

### 硬件要求
- **内存**: 最低 4GB RAM，推荐 8GB 或更多
- **存储**: 最低 500MB 可用空间
- **处理器**: 64位处理器

## 🔄 升级说明

这是首个正式版本，无需从之前版本升级。

## 🐛 已知问题

1. **文件对话框**: 当前版本移除了rfd依赖以避免async-recursion问题
2. **大文件索引**: 索引大量文件时可能需要较长时间
3. **主题切换**: 某些情况下主题切换可能需要重启应用
