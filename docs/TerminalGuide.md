# iTerminal 安全终端使用指南

## 📖 概述

iTerminal 是 SeeU Desktop 内置的安全终端模块，提供了一个受控的命令行环境。与传统终端不同，iTerminal 只允许执行预定义的安全命令，确保系统安全的同时提供必要的命令行功能。

## 🚀 快速开始

### 1. 打开终端
- 点击左侧导航栏的 🖥 图标
- 或使用快捷键进入终端模块

### 2. 基本操作
- 在输入框中输入命令
- 按 Enter 键执行命令
- 使用上下箭头键浏览命令历史
- 点击 "+" 按钮创建新标签页

### 3. 多标签页管理
- **创建标签页**: 点击 "+" 按钮
- **切换标签页**: 点击标签页标题
- **关闭标签页**: 点击标签页的 "×" 按钮
- **重命名标签页**: 右键点击标签页标题

## 📋 支持的安全命令

### 📁 文件和目录操作 (10个命令)

#### `ls` / `dir` - 目录列表
```bash
ls                    # 显示当前目录内容
ls -l                 # 详细列表格式
ls -a                 # 显示隐藏文件
ls -h                 # 人性化文件大小显示
ls /path/to/directory # 显示指定目录内容
```

#### `cd` - 目录切换
```bash
cd /path/to/directory # 切换到指定目录
cd ..                 # 返回上级目录
cd ~                  # 切换到用户主目录
cd                    # 切换到用户主目录
```

#### `pwd` - 当前目录
```bash
pwd                   # 显示当前工作目录的完整路径
```

#### `cat` / `type` - 文件内容显示
```bash
cat filename.txt      # 显示文件全部内容
type filename.txt     # Windows风格的文件内容显示
```

#### `head` / `tail` - 文件头尾显示
```bash
head filename.txt     # 显示文件前10行
head -n 20 file.txt   # 显示文件前20行
tail filename.txt     # 显示文件后10行
tail -n 5 file.txt    # 显示文件后5行
```

#### `wc` - 文件统计
```bash
wc filename.txt       # 显示行数、单词数、字符数
wc -l filename.txt    # 只显示行数
wc -w filename.txt    # 只显示单词数
wc -c filename.txt    # 只显示字符数
```

#### `find` - 文件查找
```bash
find . -name "*.txt"  # 查找当前目录下的txt文件
find /path -name file # 在指定路径查找文件
```

#### `tree` - 目录树显示
```bash
tree                  # 显示当前目录的树形结构
tree /path            # 显示指定目录的树形结构
```

#### `du` - 磁盘使用情况
```bash
du                    # 显示目录大小
du -h                 # 人性化显示大小
```

### 🔍 搜索和过滤 (2个命令)

#### `grep` - 文本搜索
```bash
grep "pattern" file.txt        # 在文件中搜索模式
grep -i "pattern" file.txt     # 忽略大小写搜索
grep -n "pattern" file.txt     # 显示行号
grep -r "pattern" directory/   # 递归搜索目录
```

#### `which` - 命令位置查找
```bash
which ls              # 查找ls命令的位置
which python          # 查找python命令的位置
```

### 💻 系统信息 (7个命令)

#### `whoami` - 当前用户
```bash
whoami                # 显示当前用户名
```

#### `hostname` - 主机名
```bash
hostname              # 显示主机名
```

#### `date` - 日期时间
```bash
date                  # 显示当前日期和时间
date -u               # 显示UTC时间
date -I               # ISO格式日期
```

#### `uptime` - 系统运行时间
```bash
uptime                # 显示系统运行时间和负载
```

#### `ps` - 进程信息
```bash
ps                    # 显示当前用户的进程
ps -a                 # 显示所有用户的进程
```

#### `env` - 环境变量
```bash
env                   # 显示所有环境变量
```

#### `df` - 文件系统使用情况
```bash
df                    # 显示文件系统使用情况
df -h                 # 人性化显示大小
```

### 🛠️ 实用工具 (5个命令)

#### `echo` - 文本输出
```bash
echo "Hello World"    # 输出文本
echo $HOME            # 输出环境变量
```

#### `clear` - 清空终端
```bash
clear                 # 清空终端屏幕
```

#### `history` - 命令历史
```bash
history               # 显示命令历史记录
```

#### `help` - 帮助信息
```bash
help                  # 显示所有可用命令
help ls               # 显示特定命令的帮助
```

#### `exit` - 退出会话
```bash
exit                  # 退出当前终端会话
```

## ⚙️ 终端设置

### 访问设置
1. 打开全局设置（点击 🔩 图标）
2. 选择"💻 终端设置"分类
3. 配置各项设置

### 外观设置

#### 字体配置
- **字体大小**: 8-24px，默认14px
- **字体缩放**: 0.5-3.0倍，默认1.0倍
- **滚动缓冲**: 100-50000行，默认1000行

#### 颜色主题
- **背景色**: 自定义终端背景颜色
- **文本色**: 自定义文本颜色
- **光标色**: 自定义光标颜色

### 行为设置

#### Shell配置
- **默认Shell**: 设置默认的Shell命令
- **工作目录**: 设置默认工作目录

#### 交互设置
- **响铃开关**: 启用/禁用终端响铃
- **光标闪烁**: 0-2000ms，默认500ms
- **Tab大小**: 2-8空格，默认4空格

### 操作功能
- **保存设置**: 保存当前配置
- **重置默认**: 恢复默认设置
- **导出配置**: 导出设置到文件（预留功能）
- **导入配置**: 从文件导入设置（预留功能）

## 📜 命令历史管理

### 查看历史
1. 点击 "📜" 按钮打开历史对话框
2. 浏览所有执行过的命令
3. 使用搜索框查找特定命令

### 使用历史
- **重新执行**: 双击历史命令重新执行
- **复制命令**: 右键复制命令到剪贴板
- **清空历史**: 清除所有历史记录

### 快捷键
- **上箭头**: 浏览上一条命令
- **下箭头**: 浏览下一条命令
- **Ctrl+R**: 搜索历史命令（计划功能）

## 🔒 安全特性

### 命令白名单
- 只允许执行预定义的23个安全命令
- 禁止执行系统级危险操作
- 防止恶意代码执行

### 参数验证
- 对命令参数进行安全检查
- 防止路径遍历攻击
- 限制文件访问范围

### 沙箱环境
- 命令在受限环境中执行
- 无法访问系统关键文件
- 无法执行网络相关操作

## 💡 使用技巧

### 高效操作
1. **Tab补全**: 使用Tab键自动补全文件名（计划功能）
2. **快捷键**: 熟练使用上下箭头浏览历史
3. **多标签**: 为不同任务创建不同标签页
4. **管道操作**: 组合使用多个命令

### 常用组合
```bash
# 查找大文件
find . -name "*.log" | head -10

# 统计代码行数
find . -name "*.rs" | wc -l

# 搜索配置文件
grep -r "config" . | head -5

# 查看系统信息
whoami && hostname && date
```

### 最佳实践
1. **定期清理**: 定期清理命令历史和输出缓冲
2. **合理命名**: 为标签页设置有意义的名称
3. **安全意识**: 不要尝试执行不在白名单中的命令
4. **备份设置**: 定期备份终端配置

## 🔧 故障排除

### 常见问题

#### 命令无法执行
- 检查命令是否在安全命令列表中
- 验证命令语法是否正确
- 确认文件路径是否存在

#### 输出显示异常
- 检查字体设置是否正确
- 调整字体大小和缩放比例
- 重置终端设置到默认值

#### 性能问题
- 减少滚动缓冲区大小
- 清理命令历史记录
- 关闭不必要的标签页

### 获取帮助
1. 使用 `help` 命令查看可用命令
2. 使用 `help 命令名` 查看特定命令帮助
3. 查看应用日志获取详细错误信息
4. 通过AI助手询问使用问题
