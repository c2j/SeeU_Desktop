# SeeU Desktop 用户手册更新总结

## 更新概述

我已经根据当前功能的实际实现情况，全面更新了 `docs/UserGuide.md` 用户手册，使其与实际功能保持一致。

## 主要更新内容

### 1. 界面布局更新

#### 原版本
- 简单的界面布局图
- 缺少具体的导航模块说明

#### 更新版本
- 详细的界面布局图，包含所有导航模块
- 新增导航栏模块说明：
  - 🏠 主页：应用启动页面，功能概览和快速访问
  - 🖥 终端：iTerminal智能终端，支持多标签页
  - 📝 笔记：iNote智能笔记系统，支持Markdown编辑
  - 🔍 搜索：iSearch全局文件搜索功能
  - 📁 文件：iFile Editor高性能文件编辑器
  - 🛠 工具：iTools AI工具集成和MCP服务器管理
  - ⚙️ 设置：应用设置和模块配置

### 2. 主页功能重写

#### 原版本
- 简单的功能卡片介绍
- 基础的快速操作说明

#### 更新版本
- **豆腐块布局设计**：详细说明左右列布局
- **左列功能模块**：笔记、文件编辑器、终端、MCP工具豆腐块
- **右列系统信息**：系统状态、数据统计、快速操作
- **全局搜索功能**：跨模块搜索、结果展示、快速跳转

### 3. iTerminal 完全重写

#### 原版本
- "安全终端"概念，只支持预定义命令
- 列出了23个具体的安全命令

#### 更新版本
- **智能终端**：基于 egui_term 和 alacritty_terminal
- **完整终端支持**：
  - 真实终端模拟，支持所有系统命令
  - GPU 加速渲染
  - 完整 Shell 支持（bash、zsh、PowerShell等）
  - ANSI 颜色支持
- **会话管理**：新建、切换、关闭会话，会话历史
- **高级功能**：
  - 会话历史管理和恢复
  - 导出功能（文本、Markdown）
  - 内置帮助系统
  - 详细的终端设置

### 4. 新增 iFile Editor 章节

#### 全新内容
- **ROPE算法支持**：大文件编辑、高效操作、内存友好
- **文件树功能**：专业树形视图、懒加载、多选支持
- **编辑器特性**：语法高亮、多文件编辑、智能缩进
- **使用方法**：工作区设置、文件操作、编辑功能
- **快捷键支持**：完整的快捷键列表
- **高级功能**：大文件处理、搜索集成、设置配置

### 5. 设置系统重构

#### 原版本
- 简单的模块设置列表
- 基础的配置项说明

#### 更新版本
- **模块化设置架构**：每个模块独立设置页面
- **详细设置分类**：
  - iTerminal 设置：外观、行为、颜色、操作
  - iFile Editor 设置：编辑器、文件处理、性能、快捷键
  - iNote 设置：编辑器、导入导出、存储、界面
  - iSearch 设置：搜索选项、索引、排序、性能
  - iTools 设置：MCP服务器、插件、角色、安全
  - AI助手设置：API、模型、MCP集成、对话管理
- **设置管理特性**：统一界面、配置管理、备份恢复

### 6. 常见问题扩展

#### 新增问题
- **文件编辑器问题**：大文件打开、性能优化
- **终端问题**：命令执行失败、环境配置
- **更详细的解决方案**：具体的排查步骤
