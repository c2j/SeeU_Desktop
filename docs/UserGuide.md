## 🚀 快速开始

### 安装与启动

1. **下载应用**: 从官方渠道下载适合您操作系统的版本
2. **安装运行**: 
   - Windows: 运行 `seeu_desktop.exe`
   - macOS: 打开 `SeeU Desktop.app`
   - Linux: 运行 `./seeu_desktop`
3. **首次启动**: 应用会显示主页，展示所有可用功能

## 🖥 iTerminal - 智能终端

### 基本功能

#### 现代化终端体验
SeeU Desktop 基于 egui_term 和 alacritty_terminal 提供现代化的终端体验，支持：
- **真实终端模拟**: 基于 Alacritty 终端引擎的完整终端功能
- **GPU 加速渲染**: 高性能文本渲染和滚动
- **完整 Shell 支持**: 支持系统默认 Shell（bash、zsh、PowerShell等）
- **ANSI 颜色支持**: 完整的颜色和格式化支持

#### 会话管理
- **新建会话**: 点击 "New Session" 按钮创建新的终端会话
- **会话切换**: 通过标签页或会话列表切换不同会话
- **关闭会话**: 点击 "Close Session" 按钮关闭当前会话
- **会话历史**: 支持会话历史记录和恢复功能

- **管道和重定向**: 完整支持 Shell 管道、重定向等高级功能
- **环境变量**: 完整的环境变量支持和管理
- **脚本执行**: 支持执行 Shell 脚本和批处理文件

#### 🎨 显示特性

- **颜色支持**: 完整的 ANSI 颜色和格式化支持
- **字体配置**: 可配置字体大小、字体族和缩放比例
- **主题支持**: 支持亮色和暗色主题切换

#### ⚡ 性能优化
- **GPU 渲染**: 基于 GPU 的高性能文本渲染
- **虚拟滚动**: 大量输出时的高效滚动处理
- **内存管理**: 智能的滚动缓冲区管理

- **大文件支持**: 支持编辑GB级大文件，无性能损失

#### 文件树功能

- **键盘导航**: 方向键、Enter、Shift/Ctrl组合键操作

#### 编辑器特性

## 🔍 iSearch - 全局搜索

### 基本搜索

#### 添加索引目录
1. 进入iSearch模块
2. 点击"添加目录"按钮
3. 选择要索引的文件夹
4. 等待索引完成

#### 执行搜索
1. 在搜索框中输入关键词
2. 按回车键执行搜索
3. 查看搜索结果列表

### 高级搜索语法

#### 基本语法
- `关键词`: 基本文本搜索
- `"精确短语"`: 精确匹配短语
- `+必须词`: 结果必须包含该词
- `-排除词`: 结果不能包含该词

#### 文件类型过滤
- `filetype:pdf`: 只搜索PDF文件
- `filetype:doc`: 只搜索Word文档
- `filetype:txt`: 只搜索文本文件

#### 组合查询
```
project +important filetype:pdf "quarterly report"
```
搜索包含"project"和"important"的PDF文件，且包含"quarterly report"短语

### 搜索结果

#### 结果显示
- **文件信息**: 文件名、路径、大小、修改时间
- **内容预览**: 匹配内容的上下文
- **关键词高亮**: 搜索词在结果中突出显示
- **文件图标**: 根据文件类型显示对应图标

#### 排序功能
支持多种排序方式：
- **相关性排序**: 按搜索匹配度排序（默认）
- **文件名排序**: 按文件名字母顺序排序
- **目录名排序**: 按目录路径排序
- **文件大小排序**: 按文件大小排序
- **修改时间排序**: 按文件修改时间排序

每种排序都支持升序/降序切换，点击相同排序条件可切换排序方向。

#### 结果操作
- **右键菜单**: 提供文件属性、复制路径等操作
- **快速打开**: 双击文件直接打开
- **文件夹定位**: 快速打开文件所在文件夹
- **复制功能**: 复制文件名、路径或完整路径

#### 界面优化
- **目录面板**: 默认折叠，可按需展开查看索引目录
- **全宽度布局**: 充分利用界面空间显示搜索结果
- **内容预览**: 可在设置中启用/禁用内容预览功能

## 🛠 iTools - AI工具集成

### 角色系统

#### 角色类型
- **业务用户**: 数据分析、文档处理工具
- **开发者**: 代码工具、API调试器

2. 界面会根据角色显示相应功能
3. 权限和可用插件会相应调整

### MCP服务器管理

#### 什么是MCP
MCP (Model Context Protocol) 是一个标准协议，允许AI助手与外部工具和资源进行安全交互。

#### 添加MCP服务器
1. 进入iTools模块的"MCP服务器"标签
2. 点击"添加服务器"按钮
3. 选择输入方式：
   - **表单输入**: 适合新手用户的图形化配置
   - **JSON输入**: 适合高级用户的快速配置
4. 填写服务器信息：
   - 服务器名称和描述
   - 传输类型（stdio/SSE）
   - 命令和参数
   - 工作目录和环境变量

#### 服务器状态管理
- **🔴 红灯**: 新添加或配置修改的服务器
- **🟡 黄灯**: 连接成功但未测试功能的服务器
- **🟢 绿灯**: 功能测试通过，可用于AI助手的服务器

#### 服务器测试
1. 点击服务器列表中的"🧪 测试"按钮
2. 系统会验证服务器连接和功能
3. 测试通过后状态变为绿灯
4. 查看详细的测试日志和能力信息

#### 预设服务器模板
提供7个分类的服务器模板：
- **开发工具**: Git Server、Everything Server
- **文件系统**: File System Server
- **数据库**: SQLite Server
- **Web服务**: Fetch Server
- **AI工具**: Brave Search Server
- **生产力工具**: Memory Server
- **系统工具**: 各种系统集成工具

### 插件管理

#### 浏览插件市场
1. 进入iTools模块
2. 点击"插件市场"标签
3. 浏览可用插件列表
4. 使用搜索和分类筛选

#### 安装插件

- **工具调用**: 支持AI助手调用MCP工具

## ❓ 常见问题

### 安装和启动

**Q: 应用启动缓慢怎么办？**
A: 首次启动需要初始化数据库和索引，后续启动会更快。可以在设置中启用启动优化。

**Q: 在Linux上无法启动？**
A: 确保安装了必要的系统依赖，如libgtk-3-dev、libclang-dev等。

### 功能使用

**Q: 搜索没有结果？**
A: 
1. 确认已添加并索引了目录
2. 检查搜索关键词拼写
3. 尝试使用更通用的关键词

**Q: 笔记无法保存？**
A: 
1. 检查磁盘空间是否充足
2. 确认应用有写入权限
3. 查看应用日志获取详细错误信息

**Q: AI助手无法连接？**
A:
1. 检查网络连接
2. 验证API URL和Key配置
3. 确认AI服务是否正常运行

**Q: 文件编辑器无法打开大文件？**
A:
1. 检查文件大小是否超过设置的阈值
2. 确认系统内存是否充足
3. 在设置中调整大文件处理参数
4. 尝试使用文本模式打开

**Q: 终端命令执行失败？**
A:
1. 确认命令在系统Shell中可正常执行
2. 检查环境变量和PATH设置
3. 验证工作目录权限
4. 查看终端错误输出信息

### 性能优化

**Q: 应用占用内存过多？**
A: 
1. 减少同时打开的笔记数量
2. 清理搜索缓存
3. 重启应用释放内存

**Q: 搜索速度慢？**
A: 
1. 减少索引目录数量
2. 排除不必要的文件类型
3. 定期清理索引缓存
