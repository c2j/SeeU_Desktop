## 🚀 快速开始

### 安装与启动

1. **下载应用**: 从官方渠道下载适合您操作系统的版本
2. **安装运行**: 
   - Windows: 运行 `seeu_desktop.exe`
   - macOS: 打开 `SeeU Desktop.app`
   - Linux: 运行 `./seeu_desktop`
3. **首次启动**: 应用会显示主页，展示所有可用功能

## 📝 iNote - 智能笔记

### 基本操作

#### 创建笔记本
1. 在iNote模块中，点击"新建笔记本"
2. 输入笔记本名称和描述
3. 笔记本会出现在左侧树形结构中

#### 创建笔记
1. 选择目标笔记本
2. 点击"新建笔记"按钮
3. 输入笔记标题和内容
4. 系统会自动保存

#### 编辑模式
- **Markdown模式**: 支持完整的Markdown语法
- **富文本模式**: 可视化编辑，支持格式化

### 高级功能

#### 标签管理
- 为笔记添加标签进行分类
- 使用标签云快速筛选笔记
- 支持标签的创建、编辑、删除

#### 附件支持
- 拖拽文件到笔记中添加附件
- 支持图片、文档等多种文件类型
- 附件会显示在笔记底部

#### 思源笔记导入
1. 在设置中找到"思源笔记导入"
2. 选择思源笔记的数据目录
3. 系统会自动导入笔记本、笔记和附件

### 搜索笔记
- 使用顶部搜索框搜索笔记标题和内容
- 支持全局搜索中的笔记结果
- 点击搜索结果直接跳转到笔记

## 🔍 iSearch - 全局搜索

### 基本搜索

#### 添加索引目录
1. 进入iSearch模块
2. 点击"添加目录"按钮
3. 选择要索引的文件夹
4. 等待索引完成

#### 执行搜索
1. 在搜索框中输入关键词
2. 按回车键执行搜索
3. 查看搜索结果列表

### 高级搜索语法

#### 基本语法
- `关键词`: 基本文本搜索
- `"精确短语"`: 精确匹配短语
- `+必须词`: 结果必须包含该词
- `-排除词`: 结果不能包含该词

#### 文件类型过滤
- `filetype:pdf`: 只搜索PDF文件
- `filetype:doc`: 只搜索Word文档
- `filetype:txt`: 只搜索文本文件

#### 组合查询
```
project +important filetype:pdf "quarterly report"
```
搜索包含"project"和"important"的PDF文件，且包含"quarterly report"短语

### 搜索结果
- **文件信息**: 文件名、路径、大小、修改时间
- **内容预览**: 匹配内容的上下文
- **关键词高亮**: 搜索词在结果中突出显示
- **文件图标**: 根据文件类型显示对应图标

## 🔧 iTools - AI工具集成

### 角色系统

#### 角色类型
- **业务用户**: 数据分析、文档处理工具
- **开发者**: 代码工具、API调试器
- **运维人员**: 系统监控、日志分析
- **管理员**: 完整权限管理

#### 切换角色
1. 在iTools设置中选择角色
2. 界面会根据角色显示相应功能
3. 权限和可用插件会相应调整

### 插件管理

#### 浏览插件市场
1. 进入iTools模块
2. 点击"插件市场"标签
3. 浏览可用插件列表
4. 使用搜索和分类筛选

#### 安装插件
1. 在插件市场中找到目标插件
2. 点击"安装"按钮
3. 等待安装完成
4. 插件会出现在"已安装插件"列表

#### 管理已安装插件
- 查看插件状态（运行中/已停止）
- 启用/禁用插件
- 卸载不需要的插件
- 查看插件详细信息

## 🤖 AI助手

### 基本使用

#### 开启AI助手
- 点击状态栏的"🤖 AI助手"按钮
- 右侧边栏会显示AI助手界面

#### 发送消息
1. 在输入框中输入问题或指令
2. 点击发送按钮（✈）或按回车键
3. AI会回复您的问题

### 高级功能

#### 历史会话
- 点击"历史"下拉菜单查看过往会话
- 创建新会话进行不同主题的对话
- 会话会自动保存和恢复

#### 模型设置
1. 点击设置按钮（⚙）
2. 配置API URL和API Key
3. 选择AI模型（支持OpenAI兼容接口）
4. 调整温度、最大令牌等参数

#### 附件功能
- 📎 按钮: 上传图片或文件
- @ 按钮: 使用命令菜单
- 支持代码片段分享

#### 命令菜单
使用@命令快速调用功能：
- `@文件`: 文件相关操作
- `@代码`: 代码分析和生成
- `@搜索`: 搜索相关功能
- `@数据`: 数据分析工具

## ❓ 常见问题

### 安装和启动

**Q: 应用启动缓慢怎么办？**
A: 首次启动需要初始化数据库和索引，后续启动会更快。可以在设置中启用启动优化。

**Q: 在Linux上无法启动？**
A: 确保安装了必要的系统依赖，如libgtk-3-dev、libclang-dev等。

### 功能使用

**Q: 搜索没有结果？**
A: 
1. 确认已添加并索引了目录
2. 检查搜索关键词拼写
3. 尝试使用更通用的关键词

**Q: 笔记无法保存？**
A: 
1. 检查磁盘空间是否充足
2. 确认应用有写入权限
3. 查看应用日志获取详细错误信息

**Q: AI助手无法连接？**
A: 
1. 检查网络连接
2. 验证API URL和Key配置
3. 确认AI服务是否正常运行

### 性能优化

**Q: 应用占用内存过多？**
A: 
1. 减少同时打开的笔记数量
2. 清理搜索缓存
3. 重启应用释放内存

**Q: 搜索速度慢？**
A: 
1. 减少索引目录数量
2. 排除不必要的文件类型
3. 定期清理索引缓存
