# Markdown 帮助按钮实现文档

## 概述

在笔记编辑界面的按钮栏中添加了一个"？"帮助按钮，点击后显示 Markdown 编辑的帮助说明。

## 实现的功能

### 1. 帮助按钮
- **位置**: 笔记编辑界面的工具栏中，位于富文本粘贴按钮之后
- **图标**: ❓ (问号表情符号)
- **悬停提示**: "显示 Markdown 格式指引"
- **功能**: 点击后显示 Markdown 帮助对话框

### 2. 帮助对话框
- **标题**: "Markdown 格式指引"
- **内容**: 包含完整的 Markdown 语法说明和示例
- **功能**: 
  - 显示语法示例和渲染效果对比
  - 提供复制按钮，可以直接将示例代码插入到笔记中
  - 支持滚动查看所有内容

### 3. 支持的位置
- **普通编辑模式**: 在工具栏中显示帮助按钮
- **全屏编辑模式**: 在全屏工具栏中也显示帮助按钮

## 修改的文件

## 状态管理

帮助对话框的显示状态通过 `DbINoteState` 中的 `show_markdown_help` 字段管理：

```rust
pub show_markdown_help: bool,    // Whether to show the Markdown help popup
```

- **初始状态**: `false` (对话框关闭)
- **点击帮助按钮**: 设置为 `true` (显示对话框)
- **关闭对话框**: 设置为 `false` (隐藏对话框)
