# MCP Function Calling 使用指南

## 概述

本指南介绍如何在SeeU Desktop中使用MCP (Model Context Protocol) Function Calling功能，让AI助手能够调用外部工具和服务。

## 功能特点

### ✅ 已实现的核心功能

1. **MCP服务器状态管理**
   - 🔴 红灯：服务器配置已添加/修改，需要测试
   - 🟡 黄灯：服务器已连接，需要测试功能
   - 🟢 绿灯：服务器已测试通过，可以使用

2. **AI助手集成**
   - 只有绿灯状态的MCP服务器才会出现在AI助手中
   - 自动转换MCP工具为OpenAI Function Calling格式
   - 用户确认机制确保安全性

3. **工具调用流程**
   - 检测LLM返回的工具调用请求
   - 显示详细的确认对话框
   - 执行用户确认的工具调用
   - 返回结果给LLM继续对话

## 使用步骤

### 第一步：配置MCP服务器

1. 打开SeeU Desktop应用程序
2. 切换到 **iTools** 工作区
3. 点击 **MCP设置** 标签页
4. 添加新的MCP服务器：
   - 点击 **添加MCP服务器** 按钮
   - 填写服务器配置信息
   - 选择合适的传输类型（stdio/sse）
   - 保存配置

### 第二步：测试MCP服务器

1. 在MCP设置页面找到您添加的服务器
2. 服务器状态应该显示为 🔴 红灯（需要测试）
3. 点击 **🧪 测试** 按钮进行功能测试
4. 如果测试成功，状态会变为 🟢 绿灯
5. 绿灯状态表示服务器已准备好用于AI助手

### 第三步：在AI助手中使用

1. 切换到 **AI助手** 工作区
2. 在输入框右侧找到 **MCP** 下拉菜单
3. 选择一个绿灯状态的MCP服务器
4. 发送需要工具支持的消息给AI助手

### 第四步：确认工具调用

1. 当AI助手需要调用工具时，会显示确认对话框
2. 对话框中会显示：
   - 工具名称和参数
   - 服务器信息
   - 调用类型（工具/资源/提示）
3. 仔细检查后点击 **✅ 确认执行** 或 **❌ 拒绝执行**

## 界面说明

### MCP服务器选择器

在AI助手输入框右侧，您会看到：
- **MCP:** 标签
- 下拉菜单显示可用的绿灯服务器
- 格式：`🟢 服务器ID (工具:X 资源:Y 提示:Z)`

### MCP状态面板

当选择了MCP服务器后，会显示可折叠的状态面板：
- 服务器ID和状态信息
- 可用工具、资源、提示的统计
- 详细的工具列表和描述

### 工具调用确认对话框

包含以下信息：
- 工具调用总数
- 每个工具的详细信息：
  - 工具名称
  - 服务器名称
  - 参数内容
  - 调用类型
- 安全警告和确认按钮

## 安全特性

### 用户确认机制
- 所有工具调用都需要用户明确确认
- 显示详细的调用信息供用户审查
- 支持批量确认或单独拒绝

### 状态验证
- 只有通过测试的绿灯服务器才能使用
- 自动同步服务器状态到AI助手
- 实时更新服务器可用性

## 示例场景

### 场景1：文件操作
```
用户: "请帮我列出当前目录下的所有文件"
AI: 检测到需要调用文件系统工具
系统: 显示确认对话框
用户: 确认执行
AI: 返回文件列表结果
```

### 场景2：数据查询
```
用户: "查询数据库中的用户信息"
AI: 检测到需要调用数据库工具
系统: 显示确认对话框（包含SQL查询参数）
用户: 确认执行
AI: 返回查询结果
```

## 故障排除

### 问题1：MCP服务器不显示绿灯
**解决方案：**
1. 检查服务器配置是否正确
2. 运行功能测试
3. 查看测试输出和错误信息
4. 确保服务器进程可以正常启动

### 问题2：AI助手中看不到MCP服务器
**解决方案：**
1. 确保服务器状态为绿灯
2. 检查服务器是否有可用的工具/资源/提示
3. 等待自动同步（每30秒）或重启应用

### 问题3：工具调用失败
**解决方案：**
1. 检查工具参数是否正确
2. 验证服务器连接状态
3. 查看详细的错误日志
4. 重新测试服务器功能

## 最佳实践

1. **定期测试服务器**：确保MCP服务器保持绿灯状态
2. **仔细审查工具调用**：在确认前检查所有参数
3. **监控服务器状态**：注意状态变化和错误信息
4. **合理使用工具**：只在必要时启用MCP功能

## 技术限制

1. **当前实现使用占位符执行器**：需要集成真实的MCP客户端
2. **不支持流式响应**：工具调用暂不支持流式处理
3. **单服务器选择**：一次只能选择一个MCP服务器

这个功能为SeeU Desktop提供了强大的扩展能力，使AI助手能够与外部系统安全地交互！🚀
