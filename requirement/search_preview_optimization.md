# SeeU Desktop 搜索预览功能优化

## 优化概述

本次优化主要针对搜索结果预览功能，实现了智能的关键词相关内容提取和高亮显示，使搜索结果更加精准和用户友好。

## 核心改进

### 1. 智能内容片段提取

#### 原有问题
- 搜索结果预览只显示文件开头的固定内容
- 预览内容与搜索关键词无关
- 用户无法快速判断文件是否包含所需信息

#### 优化方案
- **关键词匹配算法**: 智能查找包含搜索关键词的文本片段
- **上下文提取**: 提取关键词周围的相关上下文
- **多片段组合**: 支持显示多个不连续的匹配片段
- **相关性评分**: 根据匹配度优先显示最相关的内容

### 2. 高级搜索词解析

#### 功能特性
- **引号短语支持**: `"hello world"` 作为完整短语搜索
- **操作符过滤**: 自动过滤 `filetype:`、`filename:`、`+`、`-` 等操作符
- **短词过滤**: 忽略长度小于3的无意义词汇
- **大小写不敏感**: 自动转换为小写进行匹配

#### 实现示例
```rust
// 输入: "Rust 安全" filetype:rs +memory -unsafe
// 输出: ["rust", "安全", "memory"]

// 输入: "hello world" test
// 输出: ["hello world", "test"]
```

### 3. 智能关键词高亮

#### 高亮策略
- **Unicode安全**: 使用字符级索引，正确处理中文等多字节字符
- **重叠处理**: 智能处理重叠的搜索词，优先显示长词
- **视觉标识**: 使用 `【】` 符号标记关键词（可扩展为富文本）

#### 高亮效果示例
```
原文: "Rust是一种系统编程语言，专注于安全、速度和并发"
搜索: "Rust 安全 并发"
结果: "【Rust】是一种系统编程语言，专注于【安全】、速度和【并发】"
```

### 4. 上下文感知的片段提取

#### 智能边界检测
- **词边界识别**: 在单词边界处截断，避免截断单词
- **句子完整性**: 优先保持句子的完整性
- **段落感知**: 识别段落结构，提供更好的上下文

#### 片段组合策略
- **就近原则**: 优先组合相近的匹配片段
- **长度控制**: 根据显示空间智能调整片段长度
- **省略号处理**: 在适当位置添加省略号表示内容截断
