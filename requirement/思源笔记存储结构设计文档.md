# 思源笔记存储结构设计详解

## 概述

思源笔记采用基于文件系统的存储架构，结合SQLite数据库进行索引和搜索。数据主要存储在用户指定的工作空间目录中，通常位于 `~/SiYuan` 或用户自定义路径。

## 目录结构

### 工作空间根目录结构
```
工作空间目录/
├── assets/                 # 全局资源文件
├── emojis/                 # 自定义表情图片
├── snippets/               # 代码片段
├── storage/                # 查询条件、布局、闪卡数据等
├── templates/              # 模板片段
├── widgets/                # 挂件
├── plugins/                # 插件
├── public/                 # 公开数据
├── [笔记本ID]/             # 用户创建的笔记本文件夹
│   ├── .siyuan/
│   │   └── conf.json       # 笔记本配置
│   ├── assets/             # 笔记本专属资源
│   ├── storage/            # 笔记本专属存储
│   ├── [文档ID].sy         # 文档文件（JSON格式）
│   └── [子目录]/           # 子文档目录
└── .siyuan/                # 全局配置目录
```

### 笔记本ID规则
- 笔记本ID采用时间戳格式：`YYYYMMDDHHMMSS-随机字符`
- 例如：`20210808180117-czj9bvb`
- 必须符合正则表达式：`^[0-9]{14}-[0-9a-z]{7}$`

## 笔记本配置结构

### .siyuan/conf.json
每个笔记本都有独立的配置文件：

```json
{
  "name": "笔记本名称",
  "sort": 1,
  "icon": "1f4d4",
  "closed": false,
  "refCreateSaveBox": "",
  "refCreateSavePath": "",
  "docCreateSaveBox": "",
  "docCreateSavePath": "",
  "dailyNoteSavePath": "/daily note/{{now | date \"2006/01\"}}/{{now | date \"2006-01-02\"}}",
  "dailyNoteTemplatePath": "",
  "sortMode": 15
}
```

**字段说明：**
- `name`: 笔记本显示名称
- `sort`: 排序权重
- `icon`: 图标（Unicode编码）
- `closed`: 是否关闭状态
- `sortMode`: 排序模式（15为文件树排序）

## 文档存储格式

### .sy文件结构
文档以JSON格式存储，文件名为 `[文档ID].sy`：

```json
{
  "ID": "20200812220555-lj3enxa",
  "Spec": "1",
  "Type": "NodeDocument",
  "Properties": {
    "icon": "1f389",
    "id": "20200812220555-lj3enxa",
    "title": "文档标题",
    "type": "doc",
    "updated": "20241125224159"
  },
  "Children": [
    {
      "ID": "20241125224159-8zf3bos",
      "Type": "NodeParagraph",
      "Properties": {
        "id": "20241125224159-8zf3bos",
        "updated": "20241125224159"
      },
      "Children": [
        {
          "Type": "NodeText",
          "Data": "文本内容"
        }
      ]
    }
  ]
}
```

### 文档ID和路径规则
- 文档ID格式：`YYYYMMDDHHMMSS-随机字符`
- 文件路径：`/[文档ID].sy`
- 子文档路径：`/[父文档ID]/[子文档ID].sy`

### 节点类型（Type）
常见的节点类型包括：
- `NodeDocument`: 文档根节点
- `NodeParagraph`: 段落
- `NodeHeading`: 标题（HeadingLevel: 1-6）
- `NodeList`: 列表
- `NodeListItem`: 列表项
- `NodeBlockquote`: 引用块
- `NodeCodeBlock`: 代码块
- `NodeSuperBlock`: 超级块
- `NodeText`: 纯文本
- `NodeTextMark`: 文本标记（加粗、斜体、链接等）

## 实现示例代码

### Go语言实现示例

```go
package main

import (
    "encoding/json"
    "fmt"
    "io/fs"
    "os"
    "path/filepath"
    "regexp"
    "strings"
)

// 笔记本结构
type Notebook struct {
    ID   string `json:"id"`
    Name string `json:"name"`
    Icon string `json:"icon"`
    Path string `json:"path"`
}

// 文档结构
type Document struct {
    ID         string                 `json:"ID"`
    Spec       string                 `json:"Spec"`
    Type       string                 `json:"Type"`
    Properties map[string]interface{} `json:"Properties"`
    Children   []Node                 `json:"Children"`
}

// 节点结构
type Node struct {
    ID           string                 `json:"ID,omitempty"`
    Type         string                 `json:"Type"`
    Data         string                 `json:"Data,omitempty"`
    Properties   map[string]interface{} `json:"Properties,omitempty"`
    Children     []Node                 `json:"Children,omitempty"`
    HeadingLevel int                    `json:"HeadingLevel,omitempty"`
}

// 读取工作空间中的所有笔记本
func ReadNotebooks(workspaceDir string) ([]Notebook, error) {
    var notebooks []Notebook
    idPattern := regexp.MustCompile(`^[0-9]{14}-[0-9a-z]{7}$`)

    entries, err := os.ReadDir(workspaceDir)
    if err != nil {
        return nil, err
    }

    for _, entry := range entries {
        if !entry.IsDir() {
            continue
        }

        if !idPattern.MatchString(entry.Name()) {
            continue
        }

        confPath := filepath.Join(workspaceDir, entry.Name(), ".siyuan", "conf.json")
        if _, err := os.Stat(confPath); os.IsNotExist(err) {
            continue
        }

        confData, err := os.ReadFile(confPath)
        if err != nil {
            continue
        }

        var conf map[string]interface{}
        if err := json.Unmarshal(confData, &conf); err != nil {
            continue
        }

        notebook := Notebook{
            ID:   entry.Name(),
            Name: conf["name"].(string),
            Icon: conf["icon"].(string),
            Path: filepath.Join(workspaceDir, entry.Name()),
        }

        notebooks = append(notebooks, notebook)
    }

    return notebooks, nil
}

// 读取笔记本中的所有文档
func ReadDocuments(notebookPath string) ([]Document, error) {
    var documents []Document

    err := filepath.WalkDir(notebookPath, func(path string, d fs.DirEntry, err error) error {
        if err != nil {
            return err
        }

        if !strings.HasSuffix(path, ".sy") {
            return nil
        }

        data, err := os.ReadFile(path)
        if err != nil {
            return err
        }

        var doc Document
        if err := json.Unmarshal(data, &doc); err != nil {
            return err
        }

        documents = append(documents, doc)
        return nil
    })

    return documents, err
}

// 提取文档的纯文本内容
func ExtractText(node Node) string {
    var text strings.Builder

    if node.Type == "NodeText" {
        text.WriteString(node.Data)
    }

    for _, child := range node.Children {
        text.WriteString(ExtractText(child))
    }

    return text.String()
}

// 主函数示例
func main() {
    workspaceDir := "/path/to/siyuan/workspace"

    notebooks, err := ReadNotebooks(workspaceDir)
    if err != nil {
        fmt.Printf("Error reading notebooks: %v\n", err)
        return
    }

    for _, notebook := range notebooks {
        fmt.Printf("Notebook: %s (%s)\n", notebook.Name, notebook.ID)

        documents, err := ReadDocuments(notebook.Path)
        if err != nil {
            fmt.Printf("Error reading documents: %v\n", err)
            continue
        }

        for _, doc := range documents {
            title := doc.Properties["title"].(string)
            fmt.Printf("  Document: %s (%s)\n", title, doc.ID)

            // 提取文档内容
            var content strings.Builder
            for _, child := range doc.Children {
                content.WriteString(ExtractText(child))
            }

            fmt.Printf("    Content: %s...\n",
                strings.TrimSpace(content.String())[:min(100, len(content.String()))])
        }
    }
}

func min(a, b int) int {
    if a < b {
        return a
    }
    return b
}
```

### Python实现示例

```python
import json
import os
import re
from pathlib import Path
from typing import List, Dict, Any

class SiYuanReader:
    def __init__(self, workspace_dir: str):
        self.workspace_dir = Path(workspace_dir)
        self.id_pattern = re.compile(r'^[0-9]{14}-[0-9a-z]{7}$')

    def read_notebooks(self) -> List[Dict[str, Any]]:
        """读取所有笔记本"""
        notebooks = []

        for item in self.workspace_dir.iterdir():
            if not item.is_dir():
                continue

            if not self.id_pattern.match(item.name):
                continue

            conf_path = item / '.siyuan' / 'conf.json'
            if not conf_path.exists():
                continue

            try:
                with open(conf_path, 'r', encoding='utf-8') as f:
                    conf = json.load(f)

                notebooks.append({
                    'id': item.name,
                    'name': conf['name'],
                    'icon': conf['icon'],
                    'path': str(item),
                    'config': conf
                })
            except Exception as e:
                print(f"Error reading notebook {item.name}: {e}")
                continue

        return notebooks

    def read_documents(self, notebook_path: str) -> List[Dict[str, Any]]:
        """读取笔记本中的所有文档"""
        documents = []
        notebook_dir = Path(notebook_path)

        for sy_file in notebook_dir.rglob('*.sy'):
            try:
                with open(sy_file, 'r', encoding='utf-8') as f:
                    doc = json.load(f)

                # 计算相对路径
                rel_path = sy_file.relative_to(notebook_dir)
                doc['_file_path'] = str(rel_path)

                documents.append(doc)
            except Exception as e:
                print(f"Error reading document {sy_file}: {e}")
                continue

        return documents

    def extract_text(self, node: Dict[str, Any]) -> str:
        """递归提取节点的文本内容"""
        text_parts = []

        if node.get('Type') == 'NodeText':
            text_parts.append(node.get('Data', ''))

        for child in node.get('Children', []):
            text_parts.append(self.extract_text(child))

        return ''.join(text_parts)

    def export_to_markdown(self, notebook_id: str, output_dir: str):
        """将笔记本导出为Markdown格式"""
        notebooks = self.read_notebooks()
        notebook = next((nb for nb in notebooks if nb['id'] == notebook_id), None)

        if not notebook:
            raise ValueError(f"Notebook {notebook_id} not found")

        output_path = Path(output_dir)
        output_path.mkdir(parents=True, exist_ok=True)

        documents = self.read_documents(notebook['path'])

        for doc in documents:
            title = doc['Properties'].get('title', 'Untitled')
            content = []

            # 添加标题
            content.append(f"# {title}\n")

            # 提取内容
            for child in doc.get('Children', []):
                content.append(self.node_to_markdown(child))

            # 保存文件
            filename = f"{doc['ID']}.md"
            file_path = output_path / filename

            with open(file_path, 'w', encoding='utf-8') as f:
                f.write('\n'.join(content))

            print(f"Exported: {title} -> {filename}")

    def node_to_markdown(self, node: Dict[str, Any]) -> str:
        """将节点转换为Markdown格式"""
        node_type = node.get('Type', '')

        if node_type == 'NodeParagraph':
            return self.extract_text(node) + '\n'
        elif node_type == 'NodeHeading':
            level = node.get('HeadingLevel', 1)
            text = self.extract_text(node)
            return f"{'#' * level} {text}\n"
        elif node_type == 'NodeList':
            items = []
            for child in node.get('Children', []):
                if child.get('Type') == 'NodeListItem':
                    item_text = self.extract_text(child)
                    items.append(f"- {item_text}")
            return '\n'.join(items) + '\n'
        elif node_type == 'NodeCodeBlock':
            code = self.extract_text(node)
            return f"```\n{code}\n```\n"
        elif node_type == 'NodeBlockquote':
            text = self.extract_text(node)
            lines = text.split('\n')
            quoted = [f"> {line}" for line in lines if line.strip()]
            return '\n'.join(quoted) + '\n'
        else:
            return self.extract_text(node)

# 使用示例
if __name__ == "__main__":
    reader = SiYuanReader("/path/to/siyuan/workspace")

    # 读取所有笔记本
    notebooks = reader.read_notebooks()
    for nb in notebooks:
        print(f"Notebook: {nb['name']} ({nb['id']})")

    # 导出第一个笔记本为Markdown
    if notebooks:
        reader.export_to_markdown(notebooks[0]['id'], "./exported_markdown")
```
